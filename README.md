# VynStore 🛒

A fully functional e-commerce shopping cart built with vanilla HTML, CSS and JavaScript.

## Features

- 🛍️ Dynamic product rendering from JavaScript array
- 🔍 Real-time search filtering
- 🏷️ Category filter navigation (Audio, Wearables, Mobile, etc.)
- ➕ Add to cart with quantity tracking
- ➖ Increase / decrease quantity (auto-removes at 0)
- 🗑️ Remove items from cart
- 💰 Live total price calculation
- 🎉 Checkout success modal
- 📱 Fully responsive — mobile bottom navigation, hamburger menu
- 🔢 Cart badge updates across all elements

## Concepts Used

- Array methods: `map()`, `filter()`, `find()`, `forEach()`
- Spread operator `...`
- Event delegation
- `data-id` attributes
- Template literals with ternary operators
- DOM manipulation
- CSS Grid + Flexbox
- CSS custom properties (variables)
- Media queries
